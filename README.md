    观看了几分钟，他总算想起正事。

    左手拎着轻风剑术的秘籍，右手拎着木棍，配合着移动的步伐进行挥动，最初他只觉得动作难度较高，渐渐适应了后才发现这几个动作对体力的消耗极大。

    不得不暂时停下。

    汗如雨下，很累，但很满足。
